Mogaco

Mogaco(모각코) : 모여서 각자 코딩

keras-MNIST 손글씨 예제 실행 //참고 자료

download data from MNIST


MNIST 데이터(학습용 데이터 60000개 + 검증용 데이터 10000개)를 다운로드 받는다.

다운받은 데이터는 위와 같은 손글씨 데이터이다. (한 데이터의 pixel을 본 모습)

model의 layer


이미지 파일이 28 * 28 픽셀이므로 입력 값의 수는 28 * 28이고,
0~9까지 10개를 분류해야 하므로 softmax function으로 출력 뉴런의 수를 10개로 해준다.
출력 뉴런에 관한 내용 링크
activation funcation에 관해서는 더 공부가 필요할 것 같다.

Compile a model


이미지 분류의 정확성을 따져야 하므로, metrics=['accuracy']이다.
이 곳에 compile에 관한 내용이 잘 설명돼있다.

reshape


모델의 입력값은 2D(축이 2개)이므로 입력하기 알맞게 3D인 이미지파일을 2D로 바꿔준다.
(데이터 수, height, width) => (데이터 수, input_dim)

그리고, type을 float형으로 변환하고, 픽셀의 최대값인 255로 나눠준다. => 0~1사이의 값으로 표현
(왜 255로 나눠야 하는지 공부해오기. 입력받을 함수인 relu와 관련있어보임.)

to-categorical


0~9까지의 숫자일 확률을 클래스별로 지정해주기 위해 to_categorical()을 이용해준다.

fit


네트워크를 통해 데이터를 학습시킨다.

test network




test데이터를 통해 정확도를 확인한다.
