Web

World Wide Web의 약자

인터넷 vs 웹
- 인터넷은 컴퓨터가 서로 연결되어 통신을 주고받는, 컴퓨터끼리의 네트워크를 일컫는 말
- 웹은 그 인터넷상에 정보가 얽혀있는 무형의 정보 네트워크
- 인터넷에선 웹뿐 아니라 전자 메일과 같이 다른 서비스가 쓰이기도 하고, 웹은 그런 서비스 중 하나이다.

Front End vs Back End
- Front End : 웹에서 보여지는 영역(HTML, CSS, JavaScript, ...)
- Back End : 웹에서 보여지지 않는 영역(Server, DB, API, ...)

웹 해킹 : 보이는 영역의 구조와 반응을 토대로, 보이지 않는 영역을 공격한다.

HTML


HyperText Markup Language
웹 페이지를 위한 마크업 언어(태그 등을 이용하여 문서나 데이터의 구조를 명기하는 언어)
요소(여는태그 + 내용 + /닫는태그)들로 구성된다.

태그의 종류는 매우 많으므로 'html ~~ tag'등의 검색어로 찾아보며 공부하도록 하자.
+ 태그의 frequency를 찾아서 이용도가 높은 태그를 공부하는 것도 좋을 것 같다.(링크)

JavaScript

HTML : 웹의 내용 / CSS : 웹 디자인 / JavaScript : 웹의 동작

위처럼 html태그 안에 작성하여 alert에 문자열을 작성하여 출력할 수도 있고, (alert() == 알림창을 띄우는 함수)



script라는 이름의 태그 안에 .js(JavaScript 확장자)파일 주소를 작성하거나, 함수를 직접 정의하여 alert(함수이름)의 형태로 사용할 수도 있다.
+ script태그는 보통 속도개선을 위해 닫는 태그인 /body 직전에 작성한다.
+ 변수에 값을 저장할 수도 있다.
